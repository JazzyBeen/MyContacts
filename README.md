# MyContacts

Приложение для управления контактами на Android, разработанное с использованием современных инструментов разработки.

## Основные возможности

*   **Список контактов**: Отображение всех контактов из телефонной книги устройства.
*   **Алфавитная группировка**: Контакты автоматически группируются по первой букве имени.
*   **Быстрая навигация**: Боковая панель с алфавитом (Alphabet Scrollbar) для мгновенного перехода к нужному разделу.
*   **Поиск дубликатов**: Автоматическое определение дублирующихся контактов по номеру телефона.
*   **Удаление дубликатов**: Возможность быстрой очистки телефонной книги от повторяющихся записей одной кнопкой.
*   **Материальный дизайн**: Интерфейс выполнен в соответствии с гайдлайнами Material Design 3 с поддержкой светлой темы и адаптивных цветов.

## Стек технологий

*   **Язык**: Kotlin
*   **UI**: Jetpack Compose
*   **Архитектура**: Clean Architecture + MVVM
*   **Внедрение зависимостей**: Hilt
*   **Асинхронность**: Coroutines & Flow
*   **Компоненты Material**: Material 3

## Требуемые разрешения

Для работы приложению требуются следующие разрешения:
*   `READ_CONTACTS` — для чтения списка контактов.
*   `WRITE_CONTACTS` — для возможности удаления дубликатов.

При первом запуске приложение запросит необходимые разрешения.

## Структура проекта

Проект разделен на слои согласно принципам чистой архитектуры:
*   **data**: Реализация репозиториев и работа с Android ContentResolver.
*   **domain**: Модели данных, интерфейсы репозиториев и бизнес-логика.
*   **presentation**: UI-компоненты (Compose) и ViewModel.
*   **di**: Конфигурация Dagger Hilt.
